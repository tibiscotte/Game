# Game
> A Pokémon-style 2D game built with C++ and SDL2 incomplete... for now.

**How ​​do I edit the map?**
  - Access the "map" folder located within the "data" folder.
  - Edit the map in your text editor.
  - The graphics map is located in "map.txt" and the collision map in "collMap.txt"
  - 0 corresponds to no collision, 1 to a simple collision, and 2 to a collision possible only from above
  - Don't forget to modify line 55 of the main file

**How ​​do I modify the textures?**
  - Access the "blocks" folder located within the "data" folder to edit blocks, or the "entity" folder to edit entities.
  - Modify the desired texture.
  - It must have a resolution of 16px for blocks and 64px for entities.
  - Keep the same name or modify the code.
